挑选跌落排序

点击生成序号时同挑选排序，但是放弃序号后，之前比当前选项排序号高的选项的序号都会坍塌跌落。

比如4个选项分别排序为4，2，3，1，现在点击第二个选项，它将放弃序号2,同时第一个选项好和第三个选项的序号 将会跌落到3和2

PICK

PICK: = "pick"
  • Defined in index.d.ts:2236

挑选排序

点击哪个选项，就会给当前选项抢到当前还剩余的最前面的一个序号，然后在选项上表明排序号即可，无需调准选项的位置，再次点击已有需要的选项则取消该选项的序号。

比如：现在有10个选项，处于初始状态，可用的序号为1-10

  1. 点击第2个选项，按规则这时候抢到了序号1，代表把第2个选项排在第一。剩余可用序号为2-10
  2. 继续点击第5个选项，按规则这时候抢到了序号2，剩余可以用序号3-10
  3. 继续点击第2个选项，第2个选项，按规则放弃序号1，回到未排序状态，剩余可用序号1，3-10.
  4. 继续点击第8个选项，按规则，抢到序号1，剩余可用序号3-10。
