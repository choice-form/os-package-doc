Interface CFMenuQuestion

菜单题 基本内容请参考选择题CFSelectQuestion

特殊点:选项以下拉菜单的方式渲染出来 在菜单中点击某项后需要回调CFUIEventHandler.handleMenuClick方法

其他选项的渲染方式和选择题一样
