Interface CFEnvAuth

授权相关的配置，不同的运行平台有不同的账户系统，所以授权方式也不一样。 如果你构建的答题端有用户系统需要授权则这这个规则进行配置。 如果无账户系统，也不需授权则无需配置这个规则
