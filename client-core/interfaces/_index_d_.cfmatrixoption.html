Inherited from CFOption.inputRule

  • Defined in index.d.ts:6240

填空选项的输入规则,有些输入方式需要提供输入控件支持 有些则只需要提供普通输入框 所有方式都可以根据类型显示对应的提示(图标?,文字? UI自己决定)

count 输入字符,UI提供普通的输入框即可

date 输入日期,UI需要提供日期输入控件

dataRage 输入日期范围,UI需要提供日期输入范围控件

email 输入邮箱地址,UI提供普通输入框即可

float 输入小数,UI需要提供数值输入控件

int 输入整数,UI需要提供数值输入控件,且仅允许输入整数

mobile 输入手机号码,UI仅需提供普通输入框

postCode 输入邮政编码,UI仅需提供普通输入框

timeRange 输入时间范围,UI需要提供输入时间范围的控件

timeSpan 输入时间间隔,UI需要提供输入时间间隔的控件

link 输入网址,UI提供普通输入框即可

autoComplete 自动提示输入模式,UI需要提供自动提示输入控件,提示内容参照ICFValidateOptBase.completeGroups

dropdown 下拉列表方式输入,UI需要提供一个下拉列表选择空间,列表中要展示的项目参照ICFValidateOptBase.selectList

noValidation 无要求,UI提供简单的输入框即可
