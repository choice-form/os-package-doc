Optional getImageSize

  • getImageSize(url: string): Promise<CFImageSize>
    • Defined in index.d.ts:694

    获取某个url地址的图片的高宽信息的方法，有些答题端需要对图片进行布局控制， 需要事先知道图片的尺寸，核心在生成图片对象的时候会尝试调用这个方法添加尺寸信息。 无此需要的终端可以不配置这个方法。 如果不配置则使用默认值:无论什么图片都返回固定大小100*100

    deprecated

    这个方法目前只有老UI在使用，只为兼容性暂时保留，不建议使用，后续将移除， 这个信息应该由UI自主获取并使用，无需核心包插手

    Parameters

    • url: string

      图片地址

    Returns Promise<CFImageSize>
