Interface CFNodeLayout

每个节点自有布局与主题信息数据结构

其中来自于CFThemeBase中的属性是和CFTheme是一样的。

关于这些属性：

可以看成是对CFTheme全属性的一个覆盖， 当自己没有指定属性时，就是用全局的属性， 当自己有指定属性时，就使用自己的属性
