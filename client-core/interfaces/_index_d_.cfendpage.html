Interface CFEndPage

结束页面

基本内容请参考选择题CFSelectQuestion

特殊点:结束页面没有选项,不需要渲染.

如果由CFEndPage.qrCode属性有值,需要把这个值渲染成二维码 并提示,微信扫描这个二维码可以领取奖励.

如有自定以logo信息,也需要渲染
